NEXUS.AI
Developer API

Embed the AI Brain directly into your tools. Query company policy, trigger simulations, monitor employee training — all through a simple REST API.

REST API
Bearer Auth
~120ms latency
JSON responses
Base URL
HTTPS
https://api.nexus.ai/v2

All API requests must be made over HTTPS. Requests made over plain HTTP will fail. All request and response bodies are encoded as application/json.

Authentication
API Keys

The NEXUS.AI API uses Bearer token authentication. Pass your API key in the Authorization header on every request.

Your API Key

Keys are scoped to your company. Each key carries the permissions of the company it was issued to. Never expose keys in client-side code.

Quick Start
Query the AI Brain in 3 lines

The most common use case — ask your company's AI Brain a policy question and get a structured answer.

const response = await fetch('https://api.nexus.ai/v2/brain/query', {
  method: 'POST',
  headers: {
    'Authorization': 'Bearer nxs_live_your_api_key',
    'Content-Type': 'application/json',
  },
  body: JSON.stringify({
    question: 'What is the escalation procedure when a customer complaint is not resolved in 24 hours?',
    employee_id: 'emp_7k2m9p',  // optional — tracks per-employee queries
    language: 'en',
  }),
});

const data = await response.json();
console.log(data.answer);  // "According to SOP-12 section 4.2, escalate to..."
console.log(data.sources); // ["Customer_Complaint_SOP_v3.pdf", "HR_Policy_Manual.docx"]
import requests

response = requests.post(
    "https://api.nexus.ai/v2/brain/query",
    headers={
        "Authorization": "Bearer nxs_live_your_api_key",
        "Content-Type": "application/json",
    },
    json={
        "question": "What is the escalation procedure for unresolved complaints?",
        "employee_id": "emp_7k2m9p",
        "language": "en",
    }
)

data = response.json()
print(data["answer"])   # According to SOP-12 section 4.2...
print(data["sources"]) # ["Customer_Complaint_SOP_v3.pdf"]
curl -X POST https://api.nexus.ai/v2/brain/query \
  -H "Authorization: Bearer nxs_live_your_api_key" \
  -H "Content-Type: application/json" \
  -d '{
    "question": "What is the escalation procedure for unresolved complaints?",
    "employee_id": "emp_7k2m9p",
    "language": "en"
  }'

AI Brain
Endpoints
POST
/brain/query
Ask the AI Brain a question

Queries the company's AI Brain with a natural language question. Returns an answer grounded in uploaded SOPs and policy documents.

Body Parameters

ParameterTypeRequiredDescription
questionstringrequiredNatural language question to ask the AI Brain. Max 2000 characters.
employee_idstringoptionalTrack which employee asked the question. Logs to audit trail.
languagestringoptionalISO 639-1 language code. Defaults to "en".
contextstringoptionalAdditional context string injected into the query (e.g. current task description).
200 OK
400 Bad Request
401 Unauthorized
POST
/brain/train
Trigger retraining with new docs

Triggers an asynchronous retraining job for the company's AI Brain. Returns a job_id you can poll for status.

Body Parameters

ParameterTypeRequiredDescription
document_idsstring[]optionalSpecific document IDs to retrain on. If omitted, retrains on all company documents.
prioritystringoptional"normal" or "high". High priority uses dedicated capacity.
Response · 202 Accepted
{
  "job_id": "job_9xkm2p4q",
  "status": "queued",
  "estimated_duration_seconds": 45,
  "poll_url": "https://api.nexus.ai/v2/brain/status/job_9xkm2p4q"
}
GET
/brain/status/{job_id}
Poll training job status

Returns the current status of a training job. Poll this endpoint every few seconds until status === "complete".

Response · 200 OK
{
  "job_id": "job_9xkm2p4q",
  "status": "complete",     // queued | processing | complete | failed
  "progress": 100,
  "documents_trained": 12,
  "accuracy_score": 0.94,
  "completed_at": "2025-05-22T14:23:11Z"
}

Webhooks
Event-driven Integration

Register a HTTPS endpoint to receive real-time POST notifications when compliance events occur in NEXUS.AI.

All webhook payloads are signed with HMAC-SHA256 using your webhook secret. Always verify the X-Nexus-Signature header before processing.

Available Events

EventDescriptionFired when
violation.detectedEmployee missed a required SOP stepReal-time
simulation.completedEmployee finished a simulation sessionOn submit
simulation.failedScore fell below the configured thresholdOn submit
certificate.issuedEmployee earned full certificationOn 100%
brain.trainedAI Brain training job completedAsync
employee.createdNew employee added to the platformInstant
document.uploadedNew SOP document uploadedInstant
